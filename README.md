# 🎅 Secret Santa Game (Digital Edition)

Welcome to the Digital Secret Santa Game!  
This project brings the traditional Secret Santa experience—usually played in person—into a digital format where assignments and communication are handled via email.

## Overview

Secret Santa is a holiday tradition in which each participant is randomly assigned another person to give a gift to, keeping identities secret until the exchange. This digital version automates the process and uses email to notify participants of their assignments anonymously.

## Features

- **Easy Registration**: Participants sign up using their email addresses.
- **Random Assignment**: The system randomly matches each participant to a gift recipient, ensuring no one is assigned to themselves.
- **Email Notifications**: Each participant receives an email with the details of their assigned recipient.
- **No Need for an App**: All coordination and communication happen via email; no additional platform required.
- **Anonymous Messaging** (optional): Send wishlists or hints anonymously via the organizer's email.

## How It Works

1. **Organizer Collects Emails**: The organizer gathers emails from all participants.
2. **Run the Assignment Tool**: The digital tool randomly assigns each participant a recipient.
3. **Emails Are Sent Out**: Each participant receives an email (from the organizer’s email address) telling them who they should buy a gift for, while keeping assignments secret.
4. **Gift Exchange**: Gifts are prepared and exchanged in-person or by mail, as decided by the group.
